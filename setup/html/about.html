What?
The CPAN Cross Reference is piece of software that creates cross-references from source code. If cross-referencing is something new to you, take a look at the Linux Cross Reference.

CPAN Cross Reference doesn't use LXR, instead it uses the CPANXR module which can be found on CPAN. It is also cross-referenced here.

History
CPAN Cross Reference was born at a london.pm emergency social meeting in late july '03.
